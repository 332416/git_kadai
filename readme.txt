・やること
1.クローンしたフォルダに自分のブランチをつくる
$git branch <ブランチ名>

2.ブランチができたか確認する
$git branch

3.自分のブランチに移動する
$git checkout <1.で作ったブランチ名>

4.ブランチに移動できたか確認する
$git branch

5.VEP.txtに適当に書き加える(なんでもOK)
6.フォルダに適当なファイルをつくる(もしくは下アドレスからダウンロードしてフォルダに張る)
https://drive.google.com/file/d/1oKfwrfLjVo6CQGsyE0OpXPYa5FWIajbh/view?usp=sharing

7.変更を適応する(VEP.txt)
$git add -u

8.変更を適応する(meme.txt)
$git add meme.txt

9.コミットする
$git commit -m "text_commit"
